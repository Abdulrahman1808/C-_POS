→
2

→

4

Security & Compliance

Enterprise-Grade Security

🛡️

Row Level Security (RLS)

Every database query is filtered by business_id at the PostgreSQL level. Tenants can never see each other's data.

🔑

License-Based Authentication

Desktop app validates via unique license keys. Device limits enforced per subscription plan.

🔒

Role-Based Access Control

Granular staff permissions: inventory, reports, refunds, discounts, price changes — all individually toggled.

🔄

Offline-First with Sync Queue

Local SQLite acts as source of truth. Sync queue retries failed uploads with conflict resolution.
